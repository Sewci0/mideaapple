Flash mideaapple

Native HomeKit firmware for a Midea air conditioner on an ESP32-C3. Source & docs →

  1. Plug the ESP32-C3 into this computer over USB-C.
  2. Click Connect and pick the board's serial port.
  3. If it won't connect, hold BOOT, tap RST, release BOOT, then retry.
Your browser can't do Web Serial — use Chrome or Edge on desktop, or flash the release binary with esptool. Serial access was blocked — allow it and reload.

After flashing: open a serial monitor at 115200, type W to set WiFi, then add the accessory in the Home app with code 466-37-726. Full instructions in the README.
